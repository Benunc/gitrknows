Huntr Has a New UI

April 25, 2007 By Gitr 21 Comments

I got a new UI last night because there are some things that I have seen on the UI of the Week on WoWinsider that I really like, but I have no idea what they are. That’s when my natural instinct kicked in: work smarter, not harder. I went to Curse Gaming and found the UI I was looking for as a compilation called All Deuce UI. Here is the UI as¬† I had it when it loaded up last night. I took a shot of it this morning, but it’s not appearing on the server. I’ll do a side-by-side view for you tomorrow, as I moved the bars around and got rid of the 3D portraits.


ADUI is an all-Ace UI that is soooo easy to update with the Ace Updater, it’s not even funny. Run the updater as soon as you download it, because nearly everything had an update, but that’s all right, right? It’s Ace!

What I most loved about the set up is that you actually copy over an entire WTF folder based on your resolution, so there is no moving things around. It comes with a default setup that kicks right off the bat! I had to add Auctioneer and Fu-Experience (I couldn’t check my /played because of some chat plug-in blocking the text and Fu-XP doesn’t show time played). There were some things that¬†I was frantically trying to move, adjust, enable/disable stuff while I tried siccing my cat on stuff so I still made some progress.

The list of addons is extensive, I mean like 30-40 addons, but it’s only a¬†30MB cache file when loaded. Many could do without a number of the addons, such as sounds for chat, the 3D portraits (I turned that off and use the UF’s 3D¬†portraits), and the IM-style individual chat windows. There is a LOT to play with, and I’ve got it so it’s not being a distraction any more, and I would venture to say that it is helping me play faster and smarter. Now if only I could stop accidently targeting Rare Elites 1-4 levels above me without realizing my danger…
